A generated UI (FORMs) for JSON documents, driven by Schema.

[Live demo](https://ibm.github.io/diesel-json-form/)

Features :

- dynamic UI generation for JSON documents
- extensible via "renderers"
- schema validation / error reporting
- proposals (fills JSON with schema-valid data)
- pluggable "schema service"
